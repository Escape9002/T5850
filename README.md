# GPN tron with rust
You can find the protcol here: https://github.com/freehuntx/gpn-tron/blob/master/PROTOCOL.md


## Some tipps on how rust worked:

To run in dev-mode:
```cargo run```

To compile the release mode:
```cargo build --release``` 
The "exe" can be found under ```/target/release/```

To add dependencies:
```cargo add <package>```

## How to .env
Copy the content of ```.env.example``` into a new ```.env``` file.
Supply the correct values, compile and start the software to test it all!
